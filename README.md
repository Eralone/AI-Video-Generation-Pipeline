# AI-Video-Generation-Pipeline
VK-чат-бот принимает команды пользователя, собирает параметры (тема, фон, аватар, голос) и запускает пайплайн генерации: AI создаёт тексты и промты → VoiSpark генерирует аудио → Veo 3 создаёт видеоклип → Kinescope публикует результат → пользователь получает ссылку в VK.
